# Module datasourcex-kotest

[Kotest](https://kotest.io/) test support for [sl4jx-kotlin](https://caplin.github.io/DataSource-Extensions/reactive/sl4jx-kotlin) clients: a `MountableExtension` that runs a [Caplin Liberator](https://www.caplin.com/developer/caplin-platform/liberator/) inside a [Testcontainers](https://testcontainers.com/) container for the lifetime of a spec.

@see com.caplin.integration.kotest.LiberatorContainerExtension  

@sample samples.Samples.spec
