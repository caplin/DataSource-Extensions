# Module sl4jx-kotlin

Provides a [Kotlin Flow](https://kotlinlang.org/docs/flow.html)-based client for consuming data from a [Caplin Liberator](https://www.caplin.com/developer/caplin-platform/liberator/), wrapping the StreamLink and KeyMaster SDKs.

@see com.caplin.integration.streamlinkx.StreamLinkConnection
